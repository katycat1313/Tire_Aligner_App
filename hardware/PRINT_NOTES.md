# 3D Print Notes — Alignment Jig (3 Parts)

These files are the direct downloads from MakerWorld. Print all three parts to build the complete jig.

## Files

| File | Part | Color in Original |
|---|---|---|
| `ALIGNMATEV2p.3mf` | Main jig frame — clamps to the wheel rim face | Blue |
| `blockv2.3mf` | Clamp block — the piece the bolt passes through to press against the rim | Orange |
| `screwcap.3mf` | Bolt cap / knob — hand-tighten to lock the jig in place | Orange |

## Print Settings
- **Printer:** Creality K1C (tuned for this machine — re-slice if using a different printer)
- **Material:** PETG recommended — rigid, low warp, and holds up in a hot garage
- **Infill:** 40%+ for the main frame, 60%+ for the clamp block (it takes the most stress)
- **Layer height:** 0.2mm
- **Supports:** Check your slicer preview — the main frame may need supports depending on orientation

## Hardware Required
- **M8 bolt — 35mm** is the sweet spot. The jig body is 15mm deep and the clamp block is 5mm, so you only need enough extra length to reach the rim and tighten down. Anything over 40mm is going to stick out way too far and get in the way.
- **M8 nut** — standard hex nut, one is enough

## Assembly
1. Print all 3 parts
2. Thread the M8 nut into the clamp block
3. Thread the M8 bolt through the screwcap, then through the clamp block
4. The assembled clamp slides into the slot on the main jig frame
5. Hand-tighten against the wheel rim to lock the jig in place

## How It Works with the App
1. Clamp the jig to the wheel rim face — **not** the tire sidewall
2. Insert your phone into the tray, screen facing outward away from the truck
3. The jig holds the phone at a consistent repeatable angle relative to the wheel face
4. Open the app and follow the guided measurement steps
